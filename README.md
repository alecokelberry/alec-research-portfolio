# Alec Okelberry - Research Portfolio

![Portfolio Preview](./index.html) _(Screenshot to be added upon initial Vercel deployment)_

## Overview

Welcome to my academic research portfolio. I am currently building my foundation in software engineering as a Bachelor of Science in Computer Science student at Western Governors University (WGU). Prior to transitioning into tech, I conducted extensive academic research at Utah Valley University (UVU), investigating the physiological intersections of exercise performance and central fatigue.

This repository was specifically built from scratch using entirely vanilla web technologies to fulfill the Web Development Foundations (D276) course requirements at WGU. It serves as both a demonstration of foundational front-end skills and a historical record of my physiological research.

**Live Deployment:** [https://alec-research-portfolio.vercel.app](https://alec-research-portfolio.vercel.app)

## Built With

This project eschews modern frameworks (like React or Next.js) in favor of core web technologies to demonstrate fundamental competency:

- **Semantic HTML5** for structured, accessible content delivery.
- **Vanilla CSS3** featuring:
  - Custom Properties (CSS Variables) for a maintainable dark-mode theme.
  - Flexbox and CSS Grid for responsive, modular layouts.
  - Pure CSS keyframe animations and glassmorphism styling for a premium feel.
- **Zero Dependencies:** No external JavaScript libraries, frameworks, or build tools were used.

## Research Highlights

The portfolio highlights two significant pieces of research from my time at UVU:

1. **Investigating Caffeine Intake Prior To Exercise:** My Honors B.S. in Exercise Science Thesis analyzing the effects of personalized caffeine dosage on exercise performance and physiological stress responses (HRV/HRC).
   - _Presented at the ACSM Southwest Chapter conference and published in the International Journal of Exercise Science._
2. **Cerebral Oxygenation During Arm vs. Leg Cycling:** A collaborative study utilizing near-infrared spectroscopy (NIRS) to evaluate prefrontal cortex oxygenation patterns between arm cranking and leg cycling.
   - _Presented at the NSCA 2023 National Conference (Summer 2023)._

## Getting Started

To view this project locally, simply clone the repository and open the `index.html` file in any modern web browser.

```bash
git clone https://github.com/alecokelberry/alec-research-portfolio.git
cd alec-research-portfolio
open index.html # On Mac
# Or simply double-click index.html in your file explorer
```
